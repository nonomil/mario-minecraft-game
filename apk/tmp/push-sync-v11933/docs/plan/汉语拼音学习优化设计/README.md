## 4. 与主项目的关系

这个目录里的方案不主张：

- 做两套完全分裂的游戏代码
- 让幼儿园模式直接复用小学英语主交互

更推荐：

- 共享底层学习事件
- 共享配置驱动框架
- 分离内容层与交互层

---

## 5. 一句话结论

这个目录的核心目标不是“给小学模式补一个超低难度档”，而是：

> 给双语项目补出一个真正适合幼儿园阶段的汉字拼音学习层。
