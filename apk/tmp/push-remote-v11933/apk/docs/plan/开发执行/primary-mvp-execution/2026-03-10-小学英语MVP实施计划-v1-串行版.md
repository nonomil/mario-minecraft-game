> 执行原则：先稳住基础事件层，再接高光层，再接长期成长层

---

## 1. 目标

第一阶段只实现 3 个最小闭环：

1. 普通学习事件统一
2. BOSS / 门禁战前 `1` 题微学习
3. 学习结果 → 龙蛋成长反馈

不在本阶段做：

- 战中复杂 BOSS 提示
- 完整龙成长树
- 幼儿园模式实现

## Step 1：学习事件统一层

- 普通挑战
- 村庄训练

### 涉及模块

- `src/modules/12-challenges.js`
- `src/modules/12-village-challenges.js`
- `src/modules/10-ui.js`
- `src/modules/01-config.js`

### 子步骤

- [ ] 盘点三类学习入口
- [ ] 设计统一事件结构
- [ ] 在收词成功时产出学习事件
- [ ] 在普通挑战完成时产出学习事件
- [ ] 在村庄训练完成时产出学习事件
