## 问题与解决

### 问题1：测试初次运行全部失败

**现象：**
```
Error: resetLearningState missing
Error: triggerGateMicrolearn missing
Error: window.MMWG_TEST_API.actions.resetLearningState is not a function
```

**根因：**
- 端口 4173 上的旧测试服务器缓存了过期代码
- 浏览器加载的是合并前的旧版本 JavaScript 文件

**解决方案：**
1. 切换到新端口 4180 启动测试服务器
2. 设置环境变量 `MMWG_E2E_PORT=4180`
3. 重新运行测试，所有测试通过

**教训：**
- 代码合并后，必须确保测试服务器重启以加载最新代码
- 端口冲突时，优先使用新端口而非强制终止旧进程
- 浏览器缓存可能导致测试结果不准确
