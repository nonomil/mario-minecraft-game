# 10 - BOSS战攻略

## 1. BOSS战系统概述

### 1.1 触发机制
BOSS战通过分数阈值自动触发，触发分数为：
- 第1场：2000分 → 凋零 Wither
- 第2场：4000分 → 恶魂 Ghast
- 第3场：6000分 → 烈焰人 Blaze
- 第4场：8000分 → 凋零骷髅 Wither Skeleton

### 1.2 战场规则
- 进入BOSS战后，战场封闭
- 屏幕顶部显示BOSS血条和阶段信息
- 击败BOSS后获得丰厚奖励
- 每个BOSS只触发一次

### 1.3 通用机制
- 所有BOSS都有多阶段（Phase）设计
- 血量降低到阈值时切换阶段
- 每个阶段攻击模式不同，难度递增
- 近战和弓箭都可对BOSS造成伤害

## 2. 第一BOSS：凋零 Wither

### 2.1 基础数据
| 属性 | 值 |
|------|-----|
| 生命值 | 30 HP |
| 触发分数 | 2000 |
| 尺寸 | 96×96 |
| 基础伤害 | 1 |
| 阶段切换 | 60%血量 → 阶段2，20%血量 → 阶段3 |

### 2.2 阶段详解

#### 阶段一：警戒（100%~60% HP）
- 缓慢飘移跟随玩家
- 每3秒（180帧）发射1个黑球
- 黑球直线飞向玩家位置
- 移动速度：1.5

**攻略要点**：
- 黑球速度不快，左右移动即可躲避
- 趁发射间隙近身攻击
- 弓箭远程输出更安全

#### 阶段二：暴怒（60%~20% HP）
- 移动速度提升至×1.5
- 每2秒（120帧）发射3个扇形黑球
- 新增冲刺攻击
- 变红色视觉提示

**攻略要点**：
- 扇形弹幕需要找间隙穿过
- 冲刺前有预警，及时跳跃躲避
- 保持中距离，不要贴太近

#### 阶段三：狂暴（20%~0% HP）
- 固定在中央位置
- 全方位发射追踪弹
- 攻击频率大幅提升

**攻略要点**：
- 追踪弹有转向限制，大幅度移动可甩开
- 这是DPS竞赛阶段，全力输出
- 召唤铁傀儡分担压力

### 2.3 击败奖励
- +500 分
- +5 铁块
- 钻石盔甲

## 3. 第二BOSS：恶魂 Ghast

### 3.1 基础数据
| 属性 | 值 |
|------|-----|
| 生命值 | 25 HP |
| 触发分数 | 4000 |
| 尺寸 | 64×64 |
| 基础伤害 | 2 |
| 阶段切换 | 50%血量 → 阶段2，20%血量 → 阶段3 |

### 3.2 核心机制：火球反弹
- 恶魂发射火球
- 玩家近战攻击可将火球打回（变蓝色）
- 反弹火球命中恶魂造成3点伤害
- 这是对恶魂最高效的伤害方式

### 3.3 哭泣状态
- 累计受击10次后进入哭泣状态
- 哭泣持续5秒，期间不攻击
- 流泪粒子效果
- 是集中输出的最佳窗口

### 3.4 阶段详解

#### 阶段一（100%~50% HP）
- 8字形飘移轨迹（范围150px）
- 移动速度：0.02
- 定期发射火球

**攻略要点**：
- 学会反弹火球是关键
- 8字形轨迹可预判位置

#### 阶段二（50%~20% HP）
- 8字形范围不变
- 移动速度提升至0.03
- 火球频率增加
- 新增随机突进攻击（有预警）

**攻略要点**：
- 突进前有视觉预警，及时闪避
- 继续利用火球反弹

#### 阶段三（20%~0% HP）
- 8字形范围扩大至200px
- 移动速度提升至0.04
- 弹幕数量递增

**攻略要点**：
- 全力输出，利用每次哭泣窗口
- 弓箭远程补充伤害

## 4. 第三BOSS：烈焰人 Blaze

### 4.1 基础数据
| 属性 | 值 |
|------|-----|
| 生命值 | 28 HP |
| 触发分数 | 6000 |
| 尺寸 | 48×64 |
| 基础伤害 | 1 |
| 阶段切换 | 70%血量 → 阶段2，50%血量 → 阶段3 |

### 4.2 阶段详解

#### 阶段一（100%~70% HP）
- 空中浮动
- 每4秒（240帧）发射三连火球
- 火球间隔18帧

**攻略要点**：
- 三连火球有固定节奏，侧移躲避
- 弓箭是主要输出手段（Blaze在空中）

#### 阶段二（70%~50% HP）
- 三连火球间隔缩短至3秒（180帧）
- 新增火焰旋风：每10秒（600帧）在地面生成火柱
- 火柱造成持续伤害

**攻略要点**：
- 注意地面火柱位置，保持移动
- 在火柱间隙找安全位置输出

#### 阶段三（50%~0% HP）
- 三连火球间隔缩短至2秒（120帧）
- 火焰旋风继续
- 召唤小烈焰人（仅一次）
- 小烈焰人会额外攻击玩家

**攻略要点**：
- 优先清理小烈焰人
- 召唤铁傀儡帮忙处理小怪
- 持续对BOSS输出

## 5. 第四BOSS：凋零骷髅 Wither Skeleton

### 5.1 基础数据
| 属性 | 值 |
|------|-----|
| 生命值 | 40 HP |
| 触发分数 | 8000 |
| 尺寸 | 48×96 |
| 基础伤害 | 1 |
| 阶段切换 | 60%血量 → 阶段2，30%血量 → 阶段3 |
| 移动速度 | 2.0（地面） |
| 重力 | 0.5 |

### 5.2 核心机制
- 唯一的地面型BOSS
- 近战连击系统
- 格挡机制
- 跳跃攻击

### 5.3 阶段详解

#### 阶段一（100%~60% HP）
- 地面巡逻，追踪玩家
- 接近64px内发动近战连击
- 基础移动速度2.0

**攻略要点**：
- 保持距离，用弓箭风筝
- 连击有间隔，趁间隙反击
- 跳跃躲避近战攻击

#### 阶段二（60%~30% HP）
- 移动速度提升至×1.3
- 攻击更加频繁
- 可能使用格挡（减少受到的伤害）

**攻略要点**：
- 格挡时不要浪费弹药
- 等格挡结束再集中输出
- 召唤傀儡分担仇恨

#### 阶段三（30%~0% HP）
- 召唤小骷髅兵（仅一次）
- 全力追击玩家
- 跳跃攻击

**攻略要点**：
- 优先处理小骷髅兵
- 利用平台高度差躲避跳跃攻击
- 全力输出，这是最后阶段

## 6. BOSS战通用攻略

### 6.1 战前准备
1. 确保背包有足够箭矢（至少20支）
2. 准备食物用于回血
3. 钻石备用（紧急回血按Z）
4. 如有末影珍珠可用于紧急位移
5. 铁块×3准备召唤铁傀儡

### 6.2 战斗技巧
1. 优先学会躲避弹幕模式
2. 利用阶段切换的短暂停顿输出
3. 召唤物可以分担BOSS注意力
4. 弓箭蓄力满伤害更高
5. 近战在安全窗口使用，DPS更高

### 6.3 难度影响
BOSS战受当前难度层级影响：
- 新手难度：BOSS伤害×0.8
- 地狱难度：BOSS伤害×1.8
- DDA系统在BOSS战中同样生效
