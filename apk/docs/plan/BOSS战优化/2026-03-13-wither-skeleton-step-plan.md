# BOSS战优化 Step-by-Step 计划

## 目标
- 提升可学习性与战斗节奏
- 明确阶段化与反馈
- 增加风险/收益决策

## 验收标准（通用）
- 进入战斗后 3~5 秒内能看到明确“提示/节奏”
- 任何高伤害招式都有可识别预警
- 有至少 1 个明确的“惩罚窗口”（玩家可反击）
- 调试入口可稳定复现

## BOSS：Wither Skeleton

### 设计要点
- 突进冲刺 + 明显后摇
- 近战“躲-打”节奏

### Step-by-Step
1. 现状梳理
   - 近战攻击范围与频率
2. 新增突进冲刺
   - 0.6s 预警
   - 冲刺后停顿 1s
3. 反击窗口
   - 冲刺后硬直给玩家输出
4. 调试接口
   - 控制冲刺频率
5. 测试
   - 冲刺可以躲避
   - 后摇可打出稳定输出

### 风险点
- 冲刺频率过高导致压迫感过强
