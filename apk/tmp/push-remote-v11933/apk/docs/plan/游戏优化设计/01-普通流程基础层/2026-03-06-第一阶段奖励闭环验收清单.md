## 1. 本清单目的

本清单用于验证：

> 普通流程里的学习，是否已经从“更多弹窗题”升级为“更自然地发生、更容易承接、更不打断”的基础层学习融合。

重点不再只看“答题后是否给奖励”，而要看：

1. 收词是否有语义强化。
2. 普通流程是否减少了不必要打断。
3. 村庄是否成为低压学习中枢。
4. 宝箱 / 收藏 / 图鉴是否能承接学习结果。
5. 基础层是否与 BOSS / 龙专题边界清晰。

---

## 2. 验收范围

本轮只覆盖普通流程基础层：

- 收词
- 普通挑战
- Word Gate
- 复活学习
- 村庄训练
- 轻复盘
- 宝箱 / 收藏 / 图鉴承接

本轮不作为实现目标的内容：

- BOSS 战前微学习
- BOSS 战中技能词提示
- 龙蛋成长型学习
- 骑龙操作词汇化

它们只作为兼容性检查项存在。

## 4. 功能验收项

## 4.1 普通学习事件统一性

**目标：** 学习事件不再只围绕答题结算定义。

- [ ] 收词能被识别为普通学习事件
- [ ] 普通挑战能被识别为普通学习事件
- [ ] 村庄训练能被识别为普通学习事件
- [ ] 复盘完成能被识别为普通学习事件

**通过标准：**

- 学习相关奖励与承接不再只依赖“答题正确”单一路径

## 4.2 收词与语义强化

**目标：** 收词不只是短暂闪现，而能帮助理解当前世界。

- [ ] 收词后有轻量上下文强化
- [ ] 提示与群系 / 敌人 / 材料 / 任务存在明确关联
- [ ] 提示长度可控，不造成跑图停顿
- [ ] 不会因为连续收词造成信息轰炸
- [ ] 单条额外提示控制在 `8 ~ 18` 个汉字或同等短英文量级

**通过标准：**

- 玩家能感知“这个词在当前场景里是什么意思 / 有什么用”

## 4.3 普通流程即时反馈

**目标：** 学习结果有可见反馈，但不打断节奏。

- [ ] 轻量学习事件会给轻提示
- [ ] 关键达成事件会给更明显反馈
- [ ] 反馈不遮挡主操作区
- [ ] 同屏反馈数量受控

**通过标准：**

- 玩家知道自己得到了什么，但不会觉得普通流程变得很吵

## 4.4 宝箱 / 收藏 / 图鉴承接

**目标：** 学习结果可以进入探索奖励和长期积累系统。

- [ ] 学习结果可以影响普通流程奖励承接
- [ ] 收藏 / 图鉴进度可以被普通学习事件推动
- [ ] 玩家能理解为什么某项收藏或图鉴被解锁
- [ ] 宝箱逻辑没有因为学习绑定而失衡

**通过标准：**

- 学习结果不再只流向分数，而是能流向更长期的系统

## 4.5 村庄训练与轻复盘

**目标：** 村庄从单一挑战点升级为低压学习中枢。

- [ ] 村庄可承担训练角色
- [ ] 村庄可承担轻复盘角色
- [ ] 村庄体验不偏惩罚，不偏高压
- [ ] 玩家可以在村庄整理本段学习成果

**通过标准：**

- 玩家会把村庄理解为“学习整理和准备”的区域

## 4.6 小学低年级负担控制

**目标：** 前 5~10 分钟更容易进入状态。

- [ ] 普通流程文字量受控
- [ ] 单次提示更短
- [ ] 弹窗频率没有进一步升高
- [ ] 仍然保留基本参与和判断空间
- [ ] 普通挑战频率约收敛到 `0.18 ~ 0.22`
- [ ] 普通挑战限时约为 `18s`
- [ ] 普通挑战答错惩罚约为 `-3 ~ -4`
- [ ] 单词卡停留约为 `1000 ~ 1200ms`

**通过标准：**

- 小学低年级玩家更容易坚持下去，而不是被频繁阅读负担劝退

## 5. 回归兼容项

以下内容不属于本轮实现目标，但必须确认不被破坏：

- [ ] 现有 BOSS 战流程正常
- [ ] Biome Gate / 门禁战节奏正常
- [ ] 龙蛋召唤流程正常
- [ ] 骑龙控制与 HUD 正常
- [ ] Word Gate 正常
- [ ] 复活学习流程正常
- [ ] 宝箱生成与掉落正常
- [ ] Word Gate 前期出现频率明显低于旧默认值
- [ ] 复活学习更像恢复机会，不像二次惩罚

## 6. 推荐验收流程

建议按下面顺序验：

1. 先验收一次普通跑图收词
2. 再验收一次普通挑战与反馈
3. 再验收宝箱 / 收藏 / 图鉴承接
4. 再验收村庄训练与轻复盘
5. 最后做与 BOSS / 龙系统的兼容性回归

建议覆盖的最小场景：

- 场景 A：连续收词，检查语义强化与提示节奏
- 场景 B：完成一次普通挑战，检查普通流程反馈
- 场景 C：完成一段探索后，检查宝箱 / 收藏 / 图鉴承接
- 场景 D：进入村庄，检查训练与轻复盘
- 场景 E：再跑一次带门禁或龙相关流程的兼容回归
- 场景 F：低年级默认参数下体验前 `10` 分钟，检查挑战频率、提示密度、Word Gate 与复活学习是否温和化

## 8. 通过门槛

- [ ] 普通学习事件不再只依赖答题结算
- [ ] 收词有上下文强化
- [ ] 村庄已具备训练或复盘承接能力
- [ ] 宝箱 / 收藏 / 图鉴至少有一类承接打通
- [ ] 普通流程整体没有明显变得更吵或更打断
- [ ] BOSS / 门禁战 / 龙蛋 / 骑龙兼容正常
- [ ] 村庄一次训练可控制在约 `90s` 内
- [ ] Word Gate 默认值已温和化（建议目标：`0.12 / 240`）
- [ ] 复活学习默认值已温和化（建议目标：`4 题 / 35s / 3 题正确`）

如以上任一项不满足，本轮不应宣告完成。
