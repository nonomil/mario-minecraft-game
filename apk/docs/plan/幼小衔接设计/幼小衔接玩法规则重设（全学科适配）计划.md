# 幼小衔接玩法规则重设（全学科适配）计划

## Summary
在“幼小衔接模式”下，统一重设词语闸门、复活连线、单词书/词卡与学习挑战规则，使其按“语文/数学/英语”学科特性驱动，并与桥接词库字段（subject/module/keywords/phonics/pinyin）强一致。方案以最少新增接口为目标，仅在 bridge 模式启用。

## Implementation Changes
- **模式判定与统一词条键**
  - 以 `activeVocabPack.type === "bridge"` 或 `stage === "bridge"` 启用幼小衔接玩法分支。
  - 新增 `getWordKey(wordObj)` 与 `getWordDisplayPair(wordObj)`（优先 displayContent.id / word / en），用于去重与展示。

- **桥接显示映射（BilingualVocab 适配）**
  - 扩展 `normalizeRawWord/normalizeWordContent`：识别 `subject/module/keywords/phonics`。
  - `getDisplayContent` 在 bridge 模式按学科输出：
    - 语文：主显示按语言模式；次显示优先 `pinyin`（中文/拼音模式），英语模式可回落到 `chinese`。
    - 英语：主显示 `english`；次显示 `phonics/phonetic`（无则回退 `english`）。
    - 数学：主显示 `concept/word`；次显示 `keywords[0]` 或 `module`。

- **词语闸门（word gate）规则**
  - 闸门标题按学科显示：`语文闸门/数学闸门/英语闸门`。
  - 触发题型按学科+语言模式决定：
    - 语文：pinyin → `pinyin_to_hanzi`；chinese → `hanzi_to_pinyin` 或 `fill_blank`；bilingual/english → `listen/translate`。
    - 英语：`fill_blank/multi_blank/unscramble/listen`。
    - 数学：新增“概念-关键词”选择题（优先关键词，不足则用模块名）。

- **学习挑战池（随机挑战）**
  - `pickChallengeType` 改为“按 subject + languageMode”的候选池：
    - 语文：`pinyin_to_hanzi/hanzi_to_pinyin/pinyin_tone/listen/fill_blank`
    - 英语：`listen/fill_blank/multi_blank/unscramble/translate`
    - 数学：新增 `math_concept`（概念⇄关键词）为主，必要时回落 `listen`
  - 保持旧包逻辑不变（仅 bridge 模式切换池）。

- **复活连线（word match revive）**
  - 左右栏改为“主显示⇄次显示”：
    - 语文：拼音模式为 `拼音⇄汉字`，中文模式为 `汉字⇄拼音`。
    - 英语：`单词⇄拼读/音标`。
    - 数学：`概念⇄关键词/模块名`。
  - UI 文案更新：替换“将英文与中文拉线连对”为学科对应提示。

- **单词书/词卡/回顾**
  - `recordWordResult`、`buildSessionWordsSummary`、宝箱词卡浮字、结算回顾，全部使用 displayContent 的主次字段。
  - 兼容旧包：若 displayContent 缺失则回退 `en/zh`。

- **文档输出**
  - 在 `docs/plan/幼小衔接设计` 新增：
    - `2026-03-13-幼小衔接玩法规则-design.md`
    - `2026-03-13-幼小衔接玩法规则-implementation.md`
  - UTF‑8 BOM 写入，按 AGENTS 规则验收。

## Public API / Interfaces
- 不新增外部 API。
- 内部使用 `subject/module/keywords/phonics/pinyin` 字段做 bridge 玩法分支。
- `BilingualVocab.getDisplayContent` 行为在 bridge 模式下按学科输出主次文本。

## Test Plan
- 单元测试（新增/更新）：
  - bridge 模式下 `getDisplayContent` 的学科输出正确性
  - word match 对应配对逻辑（语文/数学/英语）
  - word gate 强制题型选择与回退路径
- Playwright Debug（更新/新增）：
  - 词语闸门触发与显示（语文/数学/英语各 1）
  - 复活连线正确展示与判定
  - 拼音模式测验 + 词书回顾显示

## Assumptions
- 仅在 `stage/type === bridge` 时启用新规则，其他词库不受影响。
- 数学条目的 `keywords` 作为次显示主来源，缺失时回退 `module`。
- 英语条目优先使用 `phonics/phonetic` 作次显示，缺失时回退 `english`。
- 所有中文文档写入使用 UTF‑8 BOM，且不移动/删除任何文件。
