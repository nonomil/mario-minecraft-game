## V2 的优势

- 村庄、骑龙、合成可以先并行创造净增量
- 高冲突模块不在第一天全部交叉编辑
- 集成收口点明确，适合多人协作

## V2 的风险

- `Game.html` 和 `src/modules/16-events.js` 仍然会在 A/C/B 间形成冲突
- 如果 Stage 0 接口没冻结，Lane D 会返工
- 集成线必须严格执行全量验证，不能只跑单 lane 自测
