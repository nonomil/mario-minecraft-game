# BOSS战优化 Step-by-Step 计划

## 目标
- 提升可学习性与战斗节奏
- 明确阶段化与反馈
- 增加风险/收益决策

## 验收标准（通用）
- 进入战斗后 3~5 秒内能看到明确“提示/节奏”
- 任何高伤害招式都有可识别预警
- 有至少 1 个明确的“惩罚窗口”（玩家可反击）
- 调试入口可稳定复现

## BOSS：Blaze

### 设计要点
- 火焰旋转阵
- 移动安全区
- 近战与远程各有风险

### Step-by-Step
1. 现状梳理
   - 火球/火焰技能构成
2. 新增旋转阵
   - 固定轨迹或定时角度旋转
   - 安全区随时间移动
3. 节奏控制
   - 旋转阵持续 4~6 秒
   - 之后进入短暂输出窗口
4. 视觉提示
   - 阵法出现预告 + 边缘高亮
5. 调试接口
   - 开关旋转阵
   - 设置安全区速度
6. 测试
   - 安全区可读且可躲
   - 输出窗口稳定出现

### 风险点
- 旋转阵遮挡视线造成迷惑
