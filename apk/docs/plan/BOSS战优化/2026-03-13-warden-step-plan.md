# BOSS战优化 Step-by-Step 计划

## 目标
- 提升可学习性与战斗节奏
- 明确阶段化与反馈
- 增加风险/收益决策

## 验收标准（通用）
- 进入战斗后 3~5 秒内能看到明确“提示/节奏”
- 任何高伤害招式都有可识别预警
- 有至少 1 个明确的“惩罚窗口”（玩家可反击）
- 调试入口可稳定复现

## BOSS：Warden

### 设计要点
- 声波锁定 + 预警节拍
- 强攻击但可识别

### Step-by-Step
1. 现状梳理
   - 声波范围与伤害
2. 预警机制
   - 声波前摇（光效/震动/提示条）
3. 可反制窗口
   - 短暂硬直或打断条件
4. 调试接口
   - 触发声波
   - 触发打断
5. 测试
   - 玩家能识别并躲避
   - 可反制动作可靠

### 风险点
- 预警过短导致秒杀感
