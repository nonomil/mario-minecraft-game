﻿# 幼小衔接玩法规则重设（全学科适配）- Design

版本：2026-03-13

## 目标
- 在幼小衔接模式下重设玩法规则，确保语文/数学/英语的显示与题型匹配词库字段。
- 维持旧词库/旧模式兼容，仅在 bridge 模式启用新规则。

## 核心规则
- 统一展示来源：使用 BilingualVocab.getDisplayContent 输出主/次字段。
- 语文：中文/拼音模式优先显示拼音与汉字配对；英语模式显示词语与汉字。
- 英语：主显示英文，次显示拼读/音标（phonics/phonetic）。
- 数学：主显示概念/词条，次显示关键词或模块名。

## 玩法重设范围
- 词语闸门：题型按学科选择；闸门标题随学科变化。
- 复活连线：左右栏使用主/次字段配对。
- 单词书/村庄挑战：按主/次字段出题与选项。
- 词卡/回顾：统一使用主/次字段展示。

## 挑战题型池（bridge）
- 语文：pinyin_to_hanzi / hanzi_to_pinyin / pinyin_tone / listen / fill_blank
- 英语：listen / fill_blank / multi_blank / unscramble / translate
- 数学：math_concept

## 兼容策略
- 仅在 stage/type === bridge 启用新规则。
- 非 bridge 词库不改行为。
