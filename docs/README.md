# Minecraft Mario Words Game - 项目介绍

## 项目简介
这是一个专为幼儿园及小学生设计的英语单词学习游戏。游戏结合了 **Minecraft (我的世界)** 的方块风格与 **Super Mario (超级马里奥)** 的横版跳跃玩法，旨在通过趣味性的互动体验，帮助孩子们在玩耍中自然地记忆和巩固英语单词。

## 核心目标
- **寓教于乐**：将枯燥的单词练习转化为充满挑战的冒险游戏。
- **低门槛**：操作简单直观，支持键盘和触摸屏，适合多设备使用。
- **个性化**：提供丰富的设置选项，包括运动速度、学习模式、语音反馈等。

## 主要特性
- **双端支持**：完美适配 PC 键盘（方向键/空格/J/Y）与移动端触控。
- **智能词库**：支持多种分类词库，具备防止单词重复出现的智能筛选逻辑。
- **语音反馈**：内置中英文语音播报，增强听力理解。
- **离线运行**：优化了资源加载逻辑，支持直接双击 `Game.html` 运行，无需复杂的服务器环境。
- **存档功能**：自动保存用户的个性化设置和学习进度。

## 快速开始
1. 确保项目文件夹完整。
2. 直接使用浏览器打开根目录下的 [Game.html](../Game.html)。
3. 如果需要进行资源开发或调试，建议启动本地服务器（如 Python 的 `http.server`）。

## 技术栈
- 纯 HTML5 / CSS3 / JavaScript (ES6)
- Canvas API 负责游戏渲染
- Web Audio / Speech Synthesis API 负责音效与语音
- LocalStorage 负责数据持久化
