1. **12群系解锁分数表**

   森林(0) → 樱花(100) → 雪地(250) → 沙漠(400) → 蘑菇岛(550)
   → 山地(800) → 海洋(1100) → 火山(1500) → 地狱(2100)
   → 深暗(2800) → 末地(3800) → 天空(5000)

2. **间距梯度分析**
   - 前期：100-150（快速体验）
   - 中期：250-600（逐步挑战）
   - 后期：700-1200（高难度）

3. **蘑菇岛停留惩罚**
   - <3分钟：正常
   - 3-5分钟：敌人密度+50%
   - >5分钟：敌人密度+100%
