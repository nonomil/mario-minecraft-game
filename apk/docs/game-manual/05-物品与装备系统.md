# 05 - 物品与装备系统

## 1. 系统概述

物品系统承担三大职能：
1. 战斗策略扩展（武器、道具技能）
2. 生存与恢复管理（食物、钻石回血）
3. 长线成长反馈（装备、收藏、成就）

## 2. 背包系统

### 2.1 背包分类页签
| 页签 | 内容 |
|------|------|
| items | 消耗品、道具、特殊物品 |
| materials | 资源材料 |
| equipment | 武器与护甲 |

### 2.2 背包交互
- 点击食物：消耗1个，回复1点HP（3秒冷却）
- 点击护甲：装备/卸下
- 点击南瓜：召唤雪傀儡（消耗×1）
- 点击铁块：召唤铁傀儡（消耗×3）
- 满血时食物灰显不可用
- 冷却中食物半透明并显示⏳

## 3. 武器系统

### 3.1 武器一览
| 武器 | 类型 | 特点 | 获取方式 |
|------|------|------|----------|
| 石剑 stone_sword | 近战 | 标准伤害，冷却适中 | 稀有宝箱 |
| 木斧 axe | 近战 | 高伤害，冷却较长 | 初始装备 |
| 铁镐 iron_pickaxe | 工具 | 可挖掘平台与特殊地形 | 稀有/史诗宝箱 |
| 弓 bow | 远程 | 蓄力射击，消耗箭矢 | 稀有/史诗宝箱 |

### 3.2 弓箭机制
- 蓄力时间影响伤害倍率和弹道
- 每次射击消耗1支箭矢
- 箭矢来源：骷髅掉落、宝箱

- 通过背包点击装备/卸下

## 9. 经济平衡要点

1. 钻石是核心战略资源：回血(Z键)、复活、奖励兑换
2. 箭矢与远程输出绑定，控制弓箭使用节奏
3. 冷却道具避免一键通关
4. 高价值物品（龙蛋、末影珍珠）与高风险内容绑定
5. 食物提供稳定的低成本回复手段
