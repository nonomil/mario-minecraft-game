# 00 - 总览与阅读路径

## 1. 项目定位

这是一个横版动作闯关 + 英语单词学习融合游戏（v1.8.17）：

- 动作部分：平台跳跃、战斗、12个群系切换、4场Boss战、资源收集
- 学习部分：被动浸入（环境标签）、词语收集、宝箱绑定学习、群系复习、村庄挑战
- 生态扩展：村庄系统（NPC、休息、学习屋、存档石碑、群系特色建筑）

## 2. 核心设计目标

1. 把英语学习嵌入高频游戏行为，而不是单独做题页面
2. 通过12个群系变化保持长流程的新鲜感
3. 通过战斗和养成系统维持操作反馈
4. 通过词库轮换、间隔复习与单词本形成长期学习目标

## 3. 当前系统地图

1. 基础循环：移动 → 收集 → 战斗 → 开箱 → 提升分数
2. 学习循环：遇词 → 发音/展示 → 触发挑战 → 奖励/惩罚
3. 环境循环：分数增长 → 群系切换（每200分） → 物理/敌人变化
4. Boss循环：分数达阈值（2000/4000/6000/8000） → Boss战 → 丰厚奖励
5. 村庄循环：每500分 → 进入村庄 → 休息/学习/存档 → 继续冒险
