# 发布说明 v1.19.33

发布日期：2026-03-09

## 本次发布内容
- 修复 `启动游戏.bat` 打开的本地页面、离线单文件 `android-app/web/index.html`、以及 APK 产物中词库切换不一致的问题。
- 单文件构建器支持从 single-quoted manifest 路径提取词库文件，并强制内联全部本地 `src/*.js` / `words/vocabs/*.js` 脚本。
- 为 Minecraft 中级/完整词库和幼儿园汉字词库补齐浏览器兼容导出，切换后不再回退到默认词库。
- 设置保存时立即同步 `currentAccount.vocabulary.currentPack`，下次启动不会再被旧账号词库覆盖。
- 启动器增加 `dev-reset.html`，先清理旧 Service Worker 与缓存，再进入 `Game.html`。
